## 帧类型速查

### 控制帧（无数据载荷）

| 帧类型 | 代码 | 方向      | 说明      |
| ------ | ---- | --------- | --------- |
| DC2    | 0x12 | CTC→联锁 | 连接请求  |
| DC3    | 0x13 | 联锁→CTC | 连接确认  |
| ACK    | 0x06 | 双向      | 应答/心跳 |
| NACK   | 0x15 | 双向      | 否定应答  |
| VERROR | 0x10 | 双向      | 版本错误  |

### 数据帧（变长）

| 帧类型 | 代码 | 方向      | 说明                 |
| ------ | ---- | --------- | -------------------- |
| SDCI   | 0x8A | 联锁→CTC | 站场数据变化（增量） |
| SDI    | 0x85 | 联锁→CTC | 站场数据（完整快照） |
| SDIQ   | 0x6A | CTC→联锁 | 站场数据请求         |
| FIR    | 0x65 | 联锁→CTC | 故障信息报告         |
| RSR    | 0xAA | 双向      | 系统工作状态报告     |
| BCC    | 0x95 | CTC→联锁 | 按钮控制命令         |
| ACQ    | 0x75 | 联锁→CTC | 自律控制请求         |
| ACA    | 0x7A | CTC→联锁 | 自律控制同意         |
| TSQ    | 0x9A | 联锁→CTC | 时间同步请求         |
| TSD    | 0xA5 | CTC→联锁 | 时间同步数据         |

## 通信控制帧（无数据载荷）

### DC2帧格式（0x12）

| 字段     | 长度  | 固定值 | 说明                          |
| -------- | ----- | ------ | ----------------------------- |
| 帧头     | 1字节 | 0x7D   | 帧起始标记                    |
| 帧头长度 | 1字节 | 0x04   | 后续字段长度（不含CRC和帧尾） |
| 版本号   | 1字节 | 0x11   | 协议版本                      |
| 发送序号 | 1字节 | 0x00   | DC2固定为0                    |
| 确认序号 | 1字节 | 0x00   | DC2固定为0                    |
| 帧类型   | 1字节 | 0x12   | DC2标识                       |
| CRC      | 2字节 | 计算值 | 校验和                        |
| 帧尾     | 1字节 | 0x7E   | 帧结束标记                    |

**总长度：** 9字节

**用途：** 请求建立/重置通信会话
**方向：** CTC → 联锁

**握手流程：**

```
CTC                    联锁
  |-------- DC2 (0x12) ------>|
  |<------- DC3 (0x13) -----|
  |                         |
  [连接已建立]              |
```

### DC3帧格式（0x13）

| 字段     | 长度  | 固定值 | 说明         |
| -------- | ----- | ------ | ------------ |
| 帧头     | 1字节 | 0x7D   | 帧起始标记   |
| 帧头长度 | 1字节 | 0x04   | 后续字段长度 |
| 版本号   | 1字节 | 0x11   | 协议版本     |
| 发送序号 | 1字节 | 0x00   | DC3固定为0   |
| 确认序号 | 1字节 | 0x00   | DC3固定为0   |
| 帧类型   | 1字节 | 0x13   | DC3标识      |
| CRC      | 2字节 | 计算值 | 校验和       |
| 帧尾     | 1字节 | 0x7E   | 帧结束标记   |

**用途：** 响应DC2请求，确认连接建立
**方向：** 联锁 → CTC

**DC3后序号初始化：**

| 发送方            | 发送序号变量 | 确认序号变量 |
| ----------------- | ------------ | ------------ |
| DC3发送方（联锁） | 重置为1      | 重置为0      |
| DC3接收方（CTC）  | 重置为1      | 重置为0      |

### ACK帧格式（0x06）

| 字段     | 长度  | 说明                           |
| -------- | ----- | ------------------------------ |
| 帧头     | 1字节 | 0x7D                           |
| 帧头长度 | 1字节 | 0x04                           |
| 版本号   | 1字节 | 0x11                           |
| 发送序号 | 1字节 | 最后发送的数据帧序号（不自增） |
| 确认序号 | 1字节 | 最近收到的有效帧序号           |
| 帧类型   | 1字节 | 0x06                           |
| CRC      | 2字节 | 校验和                         |
| 帧尾     | 1字节 | 0x7E                           |

**用途：**

- 确认收到数据帧
- 心跳保活（空闲时每500ms发送）

**序号规则：**

| 字段     | 描述                                      |
| -------- | ----------------------------------------- |
| 发送序号 | 最后一帧数据帧的发送序号（ACK本身不自增） |
| 确认序号 | 最近收到的有效帧的发送序号                |

**心跳触发：** 无数据收发超过500ms时发送ACK

### NACK帧格式（0x15）

| 字段     | 长度  | 说明                     |
| -------- | ----- | ------------------------ |
| 帧头     | 1字节 | 0x7D                     |
| 帧头长度 | 1字节 | 0x04                     |
| 版本号   | 1字节 | 0x11                     |
| 发送序号 | 1字节 | 当前发送序号             |
| 确认序号 | 1字节 | 错误发生前的最后正常序号 |
| 帧类型   | 1字节 | 0x15                     |
| CRC      | 2字节 | 校验和                   |
| 帧尾     | 1字节 | 0x7E                     |

**用途：** CRC校验失败，请求重传
**重传要求：** 发送方必须重发完全相同的帧（相同序号，相同数据）

### VERROR帧格式（0x10）

| 字段     | 长度  | 说明         |
| -------- | ----- | ------------ |
| 帧头     | 1字节 | 0x7D         |
| 帧头长度 | 1字节 | 0x04         |
| 版本号   | 1字节 | 0x11         |
| 发送序号 | 1字节 | 当前发送序号 |
| 确认序号 | 1字节 | 当前确认序号 |
| 帧类型   | 1字节 | 0x10         |
| CRC      | 2字节 | 校验和       |
| 帧尾     | 1字节 | 0x7E         |

**用途：** 检测到版本号不匹配
**所需操作：** 立即停止通信，触发告警，需要人工干预

## 数据传送帧

### SDCI帧格式（0x8A）- 站场数据变化

| 字段     | 长度  | 说明        |
| -------- | ----- | ----------- |
| 帧头     | 1字节 | 0x7D        |
| 帧头长度 | 1字节 | 0x04        |
| 版本号   | 1字节 | 0x11        |
| 发送序号 | 1字节 | 递增值      |
| 确认序号 | 1字节 | 确认值      |
| 帧类型   | 1字节 | 0x8A        |
| 数据长度 | 2字节 | 小端序      |
| 站场数据 | N字节 | 每设备3字节 |
| CRC      | 2字节 | 校验和      |
| 帧尾     | 1字节 | 0x7E        |

**载荷结构（每设备）：**

| 字节 | 字段     | 说明                  |
| ---- | -------- | --------------------- |
| 0-1  | 设备索引 | 来自objects表，小端序 |
| 2    | 状态字节 | 设备状态值            |

**设备状态编码：** 状态字节是**位字段组合**，而非简单枚举，例如：

| 位位置        | 含义      | 描述             |
| ------------- | --------- | ---------------- |
| 位0（最低位） | 占用      | 0=空闲，1=占用   |
| 位1           | 锁闭状态  | 0=未锁闭，1=锁闭 |
| 位2           | 方向/进路 | 设备特定含义     |
| 位3           | 特殊状态  | 设备特定含义     |
| 位4-7         | 扩展状态  | 设备特定含义     |

**常见状态组合：**

| 状态值 | 二进制 | 含义         | 描述           |
| ------ | ------ | ------------ | -------------- |
| 0x00   | 0000   | 空闲且未锁闭 | 未占用，未锁闭 |
| 0x01   | 0001   | 占用且未锁闭 | 已占用，未锁闭 |
| 0x02   | 0010   | 空闲且锁闭   | 未占用，已锁闭 |
| 0x03   | 0011   | 占用且锁闭   | 已占用，已锁闭 |

**状态提取逻辑：**

```python
# 提取占用位（位0）
occupied = (state_byte >> 0) & 0x01

# 提取锁闭位（位1）
locked = (state_byte >> 1) & 0x01

# 对于只用低4位的设备（bit_offset=0）
state_value = state_byte & 0x0F

# 对于使用高4位的设备（bit_offset=4）
state_value = (state_byte >> 4) & 0x0F
```

**无岔区段处理：**

- 共享字节的区段，状态字节只包含相关的4位
- bit_offset = 0：使用低4位（位0-3）
- bit_offset = 4：使用高4位（位4-7）
- 参照zlobjects表的bit_offset字段进行映射
- 示例：若bit_offset=4，从 `(raw_state >> 4) & 0x0F`提取状态

**帧选择逻辑：**

| 条件                | 发送帧类型      |
| ------------------- | --------------- |
| SDCI长度 < SDI长度  | SDCI（增量）    |
| SDCI长度 >= SDI长度 | SDI（全量快照） |
| 初始同步            | SDI             |

**方向：** 联锁 → CTC

### SDI帧格式（0x85）- 站场完整数据

| 字段     | 长度  | 说明         |
| -------- | ----- | ------------ |
| 帧头     | 1字节 | 0x7D         |
| 帧头长度 | 1字节 | 0x04         |
| 版本号   | 1字节 | 0x11         |
| 发送序号 | 1字节 | 递增值       |
| 确认序号 | 1字节 | 确认值       |
| 帧类型   | 1字节 | 0x85         |
| 数据长度 | 2字节 | 小端序       |
| 站场数据 | N字节 | 完整字节数组 |
| CRC      | 2字节 | 校验和       |
| 帧尾     | 1字节 | 0x7E         |

**载荷：** 按设备位置索引的完整字节数组

- 字节位置 = 设备在zlobjects表中的byte_index - 1
- 每字节内位从低到高计数

**设备状态编码：** 状态字节是**位字段组合**，详见SDCI帧格式部分的详细位映射和提取逻辑

**方向：** 联锁 → CTC
**触发条件：** 双方通信建立成功 或 收到SDIQ请求 或 SDCI长度 >= SDI长度

### RSR帧格式（0xAA）- 系统工作状态报告

| 字段     | 长度  | 说明              |
| -------- | ----- | ----------------- |
| 帧头     | 1字节 | 0x7D              |
| 帧头长度 | 1字节 | 0x04              |
| 版本号   | 1字节 | 0x11              |
| 发送序号 | 1字节 | 递增值            |
| 确认序号 | 1字节 | 确认值            |
| 帧类型   | 1字节 | 0xAA              |
| 数据长度 | 2字节 | 小端序（固定：2） |
| 主备状态 | 1字节 | 状态码            |
| 控制模式 | 1字节 | 模式码            |
| CRC      | 2字节 | 校验和            |
| 帧尾     | 1字节 | 0x7E              |

**状态值定义：**

| 值   | 主/备状态 | 控制模式（CTC→联锁） | 控制模式（联锁→CTC） |
| ---- | --------- | --------------------- | --------------------- |
| 0x55 | 主机      | 同意自律              | 自律控制              |
| 0xAA | 备机      | 不同意自律            | 非常站控              |
| 0xCC | 未知      | 不确定                | 中间状态              |

**发送时机：** 状态变化时 或 连接建立时

### BCC帧格式（0x95）- 按钮控制命令

| 字段     | 长度  | 说明              |
| -------- | ----- | ----------------- |
| 帧头     | 1字节 | 0x7D              |
| 帧头长度 | 1字节 | 0x04              |
| 版本号   | 1字节 | 0x11              |
| 发送序号 | 1字节 | 递增值            |
| 确认序号 | 1字节 | 确认值            |
| 帧类型   | 1字节 | 0x95              |
| 数据长度 | 2字节 | 小端序（通常为7） |
| 命令数据 | N字节 | 7字节命令数据     |
| CRC      | 2字节 | 校验和            |
| 帧尾     | 1字节 | 0x7E              |

**命令数据结构：**

| 字节 | 字段      | 说明                     |
| ---- | --------- | ------------------------ |
| 0    | 命令类型  | 详见命令类型表           |
| 1-2  | 按钮索引1 | 第一个设备，小端序       |
| 3-4  | 按钮索引2 | 第二个设备（未使用则-1） |
| 5-6  | 按钮索引3 | 第三个设备（未使用则-1） |

**命令类型定义：**

| 命令       | 编码 | 参数                 |
| ---------- | ---- | -------------------- |
| 调车作业   | 12   | 终端，起始，变更区段 |
| 列车作业   | 14   | 终端，起始，变更区段 |
| 取消进路   | 4    | 仅起始信号机         |
| 故障解锁   | 6    | 区段设备             |
| 人工解锁   | 3    | 起始信号机           |
| 重复开放   | 5    | 起始信号机           |
| 道岔定位   | 7    | 道岔设备             |
| 道岔反位   | 8    | 道岔设备             |
| 道岔锁闭   | 9    | 道岔设备             |
| 道岔解锁   | 10   | 道岔设备             |
| 关闭信号   | 13   | 起始信号机           |
| 引导锁闭   | 16   | 起始信号机           |
| 引导总锁   | 18   | 起始信号机           |
| 引导总解锁 | 15   | 起始信号机           |

**使用规则：**

- 一个BCC帧 = 一个命令
- 进路命令之间最小间隔4秒，可调整

### TSD帧格式（0xA5）- 时间同步数据

| 字段     | 长度  | 说明              |
| -------- | ----- | ----------------- |
| 帧头     | 1字节 | 0x7D              |
| 帧头长度 | 1字节 | 0x04              |
| 版本号   | 1字节 | 0x11              |
| 发送序号 | 1字节 | 递增值            |
| 确认序号 | 1字节 | 确认值            |
| 帧类型   | 1字节 | 0xA5              |
| 数据长度 | 2字节 | 小端序（固定：7） |
| 时间数据 | 7字节 | YYYYMMDDHHMMSS    |
| CRC      | 2字节 | 校验和            |
| 帧尾     | 1字节 | 0x7E              |

**时间数据格式：**

| 字节 | 字段 | 格式   | 示例          |
| ---- | ---- | ------ | ------------- |
| 0-1  | 年份 | 小端序 | 0x07CD = 1997 |
| 2    | 月份 | 1字节  | 1-12          |
| 3    | 日期 | 1字节  | 1-31          |
| 4    | 小时 | 1字节  | 0-23          |
| 5    | 分钟 | 1字节  | 0-59          |
| 6    | 秒   | 1字节  | 0-59          |

**总长度：** 18字节

**示例：** `CD 07 06 15 0A 20 37` = 1997-06-21 10:32:55

### FIR帧格式（0x65）- 故障信息报告

| 字段     | 长度  | 说明          |
| -------- | ----- | ------------- |
| 帧头     | 1字节 | 0x7D          |
| 帧头长度 | 1字节 | 0x04          |
| 版本号   | 1字节 | 0x11          |
| 发送序号 | 1字节 | 递增值        |
| 确认序号 | 1字节 | 确认值        |
| 帧类型   | 1字节 | 0x65          |
| 数据长度 | 2字节 | 小端序（N+4） |
| 故障信息 | N字节 | 4字节故障数据 |
| CRC      | 2字节 | 校验和        |
| 帧尾     | 1字节 | 0x7E          |

**故障信息结构：**

| 字节 | 字段     | 说明                  |
| ---- | -------- | --------------------- |
| 0    | 提示码   | 固定为0               |
| 1-2  | 对象索引 | 16位有符号，小端序    |
| 3    | 故障类型 | 来自Error.sys的错误码 |

**对象索引编码：**

- 字节1：`index % 256`（低字节）
- 字节2：`index // 256`（高字节）
- 0xFFFF（-1）：无特定对象

- **区段** (0x02, 0x08-0x09, 0x0C, 0x17, 0x23)：区段占用/锁闭问题
- **道岔** (0x04, 0x06, 0x10-0x13, 0x20)：道岔（转辙机）相关故障
- **信号机** (0x05, 0x0B, 0x0F, 0x19-0x1A, 0x21)：信号灯/灯泡故障
- **进路** (0x03, 0x1B-0x1C)：进路/接近冲突
- **操作** (0x14, 0x16, 0x22)：操作员动作错误
- **通信** (0x24-0x25, 0x29-0x2A)：远程/通信链路问题
- **权限** (0x0D, 0x26-0x27)：口令/权限问题

**FIR帧分析示例：**

```
原始数据: 00 6F 00 05
- 提示码: 0x00
- 对象索引: 0x006F (111) - 设备#111
- 故障类型: 0x05 - 灯丝断丝

原始数据: 00 FF FF 06
- 提示码: 0x00
- 对象索引: 0xFFFF (-1) - 系统级
- 故障类型: 0x06 - 道岔锁闭
```

### ACQ帧格式（0x75）- 自律控制请求

| 字段     | 长度  | 说明   |
| -------- | ----- | ------ |

| 帧尾     | 1字节 | 0x7E   |

**用途：** 请求切换到自律控制模式
**方向：** 联锁 → CTC
**超时：** 5秒内需收到ACA响应

### ACA帧格式（0x7A）- 自律控制同意

| 字段     | 长度  | 说明                   |
| -------- | ----- | ---------------------- |
| 帧头     | 1字节 | 0x7D                   |

**用途：** 响应ACQ请求

**发送时机：**

- 连接建立后
- 切换为主机时
- 每小时一次（除00:00外）

## 数据转义规则

**转义规则：**

| 原始字节 | 转义后    | 说明     |
| -------- | --------- | -------- |
| 0x7D     | 0x7F 0xFD | 帧头标记 |
| 0x7E     | 0x7F 0xFE | 帧尾标记 |
| 0x7F     | 0x7F 0xFF | 转义字符 |

**处理顺序：**

- **发送时：** CRC计算 → 数据转义 → 发送
- **接收时：** 接收 → 数据反转义 → CRC校验

### A.3 FIR帧故障分析速查

**FIR帧结构：**

```
数据载荷：4字节
  字节0：提示码（固定0x00）
  字节1-2：对象索引（设备索引，小端序）
  字节3：故障类型（来自Error.sys的错误码）
```

**常见故障类型：**

| 错误码 | 中文描述     | 常见对象    | 说明             |
| ------ | ------------ | ----------- | ---------------- |
| 0x03   | 敌对进路     | 0xFFFF      | 进路冲突         |
| 0x05   | 灯丝断丝     | 信号机      | 信号灯泡故障     |
| 0x06   | 道岔锁闭     | 0xFFFF/道岔 | 道岔锁闭故障     |
| 0x10   | 双动道岔占用 | 0xFFFF      | 双动道岔区段占用 |
| 0x23   | 轨道故障     | 区段        | 轨道电路故障     |
| 0x29   | 通讯中断     | 0xFFFF      | 通信链路中断     |

参考文件：[`Error.sys`](./Error.sys) - 完整的错误码定义和描述

**分析示例：**

```
FIR数据: 00 6F 00 05
  → 对象索引: 0x006F (111)
  → 故障类型: 0x05 → 灯丝断丝
  → 结论: 设备#111信号灯泡故障

FIR数据: 00 FF FF 06
  → 对象索引: 0xFFFF (系统级)
  → 故障类型: 0x06 → 道岔锁闭
  → 结论: 系统级道岔锁闭故障
```

**FIR帧提取命令：**

```bash
# 提取所有FIR帧
awk '/帧类型=0x65/{print $2}' logfile

# 统计FIR帧数量
grep -c "帧类型=0x65" logfile

# 查看FIR帧十六进制数据
grep "Hex数据:" logfile | awk '{print $2, $NF}'
```

### A.5 设备状态解码速查

> 详细设备分析工作流请参考 [06-device-analysis.md](./06-device-analysis.md)

#### 道岔区段状态位定义

| 位 | 含义     | 值       |
| -- | -------- | -------- |
| 0  | 定位     | 1=定位   |
| 1  | 反位     | 1=反位   |
| 2  | 无表示   | 1=无表示 |
| 3  | 区段锁闭 | 1=锁闭   |
| 4  | 区段占用 | 1=占用   |
| 5  | 道岔锁闭 | 1=锁闭   |

**常见状态值：**

- `0x01`: 占用，未锁闭，定位
- `0x02`: 空闲，已锁闭，定位
- `0x19`: 占用，未锁闭，四开（定位+无表示）

#### 信号机状态位定义

| 位 | 含义                 | 值       |
| -- | -------------------- | -------- |
| 0  | 灯泡断丝             | 1=断丝   |
| 1  | 蓝灯                 | 1=蓝灯   |
| 2  | 白灯                 | 1=白灯   |
| 3  | 红灯                 | 1=红灯   |
| 4  | 绿灯                 | 1=绿灯   |
| 5  | 黄灯/双黄/引导白组合 | 1=组合灯 |
| 6  | 进路正在切换         | 1=切换中 |
| 7  | 延时解锁             | 1=延时中 |

#### 无岔区段状态位定义

**低4位模式（bit_offset=0）：**

- 位0：区段锁闭（1=锁闭）
- 位1：区段占用（1=占用）

**高4位模式（bit_offset=4）：**

- 位4：区段锁闭（1=锁闭）
- 位5：区段占用（1=占用）

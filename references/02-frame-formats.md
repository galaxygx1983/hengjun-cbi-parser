# 帧格式详解

## 通用帧结构

所有帧遵循同一结构：

```
[帧头 1B] [首部长 1B] [版本 1B] [发送序号 1B] [确认序号 1B] [帧类型 1B] [数据长度 2B] [数据载荷 N] [CRC 2B] [帧尾 1B]
```

| 字段 | 长度 | 固定值 |
|------|------|--------|
| 帧头 | 1B | 0x7D |
| 首部长 | 1B | 0x04 |
| 版本号 | 1B | 0x11 |
| 帧尾 | 1B | 0x7E |

## 帧类型速查

| 类型 | 代码 | 方向 | 数据载荷 |
|------|------|------|----------|
| DC2 | 0x12 | CTC→联锁 | 无 |
| DC3 | 0x13 | 联锁→CTC | 无 |
| ACK | 0x06 | 双向 | 无 |
| NACK | 0x15 | 双向 | 无 |
| VERROR | 0x10 | 双向 | 无 |
| SDCI | 0x8A | 联锁→CTC | 设备变化列表 |
| SDI | 0x85 | 联锁→CTC | 完整站场 |
| SDIQ | 0x6A | CTC→联锁 | 无 |
| FIR | 0x65 | 联锁→CTC | 故障列表 |
| RSR | 0xAA | 双向 | 2B状态 |
| BCC | 0x95 | CTC→联锁 | 7B命令 |
| ACQ | 0x75 | 联锁→CTC | 无 |
| ACA | 0x7A | CTC→联锁 | 1B响应 |
| TSQ | 0x9A | 联锁→CTC | 无 |
| TSD | 0xA5 | CTC→联锁 | 7B时间 |

## 常用帧格式

### SDCI帧 (0x8A) - 站场数据变化

**载荷**: 每设备3字节 = `[设备索引(2B)] [状态(1B)]`

```
7D 04 11 05 00 8A 06 00 25 10 7D 0A 7E
                                    └── 设备37, 状态0x10
```

### SDI帧 (0x85) - 站场完整数据

**载荷**: 完整字节数组，字节位置 = byte_index - 1

### FIR帧 (0x65) - 故障报告

**载荷**: 每故障4字节 = `[提示码(1B)] [对象索引(2B)] [故障类型(1B)]`

```
00 6F 00 05 → 对象111, 故障0x05(灯丝断丝)
00 FF FF 06 → 系统级, 故障0x06(道岔锁闭)
```

### BCC帧 (0x95) - 按钮命令

**载荷**: 7字节 = `[命令类型(1B)] [设备1(2B)] [设备2(2B)] [设备3(2B)]`

| 命令 | 编码 |
|------|------|
| 取消进路 | 4 |
| 故障解锁 | 6 |
| 道岔定位 | 7 |
| 道岔反位 | 8 |
| 引导锁闭 | 16 |

### RSR帧 (0xAA) - 工作状态

**载荷**: 2字节 = `[主备状态(1B)] [控制模式(1B)]`

| 值 | 主/备 | CTC模式 | 联锁模式 |
|----|--------|---------|----------|
| 0x55 | 主机 | 同意自律 | 自律控制 |
| 0xAA | 备机 | 不同意自律 | 非常站控 |
| 0xCC | 未知 | 不确定 | 中间状态 |

### TSD帧 (0xA5) - 时间同步

**载荷**: 7字节 = `[年份(2B,小端)] [月] [日] [时] [分] [秒]`

```
CD 07 06 15 0A 20 37 → 1997-06-21 10:32:55
```

## 设备状态解码

### 道岔区段 (纯数字名称)

| 位 | 含义 |
|----|------|
| 0 | 定位(1=定位) |
| 1 | 反位(1=反位) |
| 2 | 无表示(1=无表示) |
| 3 | 区段锁闭 |
| 4 | 区段占用 |
| 5 | 道岔锁闭 |

### 信号机 (D开头)

| 位 | 含义 |
|----|------|
| 0 | 灯丝断丝 |
| 1-4 | 灯色(蓝/白/红/绿) |
| 5 | 黄灯组合 |
| 6 | 进路转岔中 |
| 7 | 延时解锁 |

### 无岔区段

- bit_offset=0: 低4位 (位0=锁闭, 位1=占用)
- bit_offset=4: 高4位 (位4=锁闭, 位5=占用)

## 常见错误码

| 码 | 描述 | 类别 |
|----|------|------|
| 0x02 | 区段占用 | 区段 |
| 0x03 | 敌对进路 | 进路 |
| 0x05 | 灯丝断丝 | 信号机 |
| 0x06 | 道岔锁闭 | 道岔 |
| 0x23 | 轨道故障 | 区段 |
| 0x29 | 通讯中断 | 通信 |

## 约束

- 最大帧长: 1024字节
- ACK超时: 500ms
- 最大重试: 3次

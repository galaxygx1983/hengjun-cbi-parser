# 设备状态解码

## 设备类型

| 命名 | 类型 | 示例 |
|------|------|------|
| 纯数字 | 道岔区段 | "64", "10" |
| D+数字 | 信号机 | "D114", "D6" |
| 其他 | 无岔区段 | "DK5", "102G" |

## 道岔区段

| Bit | 含义 |
|-----|------|
| 0 | 定位 |
| 1 | 反位 |
| 2 | 无表示 |
| 3 | 区段锁闭 |
| 4 | 区段占用 |
| 5 | 道岔锁闭 |

## 信号机

| Bit | 含义 |
|-----|------|
| 0 | 灯丝断丝 |
| 1 | 蓝灯 |
| 2 | 白灯 |
| 3 | 红灯 |
| 4 | 绿灯 |
| 5 | 黄/2黄/引白 |
| 6 | 进路转岔中 |
| 7 | 延时解锁 |

## 无岔区段

| 位偏移 | Bit | 含义 |
|--------|-----|------|
| 0 | 0 | 锁闭 |
| 0 | 1 | 占用 |
| 4 | 4 | 锁闭 |
| 4 | 5 | 占用 |

## 典型状态

| 状态值 | 类型 | 含义 |
|--------|------|------|
| 0x19 | 道岔 | 定位+无表示+占用 |
| 0x01 | 道岔 | 定位+占用 |
| 0x08 | 信号机 | 红灯 |
| 0x10 | 信号机 | 绿灯 |
| 0x02 | 信号机 | 蓝灯 |

## 异常状态

| 状态值 | 可能原因 |
|--------|----------|
| 0x19 | 转辙机故障 |
| 0x07 | 位置冲突 |
| 0x01 | 灯丝断丝 |

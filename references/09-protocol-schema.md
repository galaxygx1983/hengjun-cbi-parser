# 协议Schema

## 帧类型

| 类型 | 代码 | 方向 | 说明 |
|------|------|------|------|
| DC2 | 0x12 | CTC→联锁 | 连接请求 |
| DC3 | 0x13 | 联锁→CTC | 连接确认 |
| ACK | 0x06 | 双向 | 应答 |
| NACK | 0x15 | 双向 | 否定应答 |
| SDCI | 0x8A | 联锁→CTC | 变化数据 |
| SDI | 0x85 | 联锁→CTC | 完整数据 |
| SDIQ | 0x6A | CTC→联锁 | 数据请求 |
| FIR | 0x65 | 联锁→CTC | 故障报告 |
| RSR | 0xAA | 双向 | 状态报告 |
| BCC | 0x95 | CTC→联锁 | 控制命令 |
| ACQ | 0x75 | 联锁→CTC | 自律请求 |
| ACA | 0x7A | CTC→联锁 | 自律响应 |
| TSQ | 0x9A | 联锁→CTC | 时间请求 |
| TSD | 0xA5 | CTC→联锁 | 时间数据 |

## 帧结构

```
[帧头1B][首部1B][版本1B][发送序号1B][确认序号1B][帧类型1B][数据长度2B][数据N][CRC2B][帧尾1B]
```

## 设备类型

| 类型 | 命名 | 示例 |
|------|------|------|
| 道岔区段 | 纯数字 | "64" |
| 信号机 | D+数字 | "D114" |
| 无岔区段 | 其他 | "DK5" |

## 状态位

| 类型 | Bit | 含义 |
|------|-----|------|
| 道岔 | 0-5 | 定位/反位/无表示/锁闭/占用 |
| 信号机 | 0-7 | 灯丝/灯色/转岔/延时 |
| 无岔 | 0,1,4,5 | 锁闭/占用 |

## 错误码

| 码 | 含义 |
|-----|------|
| 0x02 | 区段占用 |
| 0x03 | 敌对进路 |
| 0x05 | 灯丝断丝 |
| 0x06 | 道岔锁闭 |
| 0x23 | 轨道故障 |

## BCC命令

| 命令 | 编码 |
|------|------|
| 取消进路 | 4 |
| 故障解锁 | 6 |
| 道岔定位 | 7 |
| 道岔反位 | 8 |
| 引导锁闭 | 16 |

## RSR状态

| 值 | 主/备 | 模式 |
|----|-------|------|
| 0x55 | 主机 | 自律/同意 |
| 0xAA | 备机 | 非常站控/不同意 |
| 0xCC | 未知 | 中间/不确定 |

## 超时参数

| 参数 | 值 |
|------|-----|
| ACK超时 | 500ms |
| 中断检测 | 1500ms |
| ACQ超时 | 5秒 |
| CTC切换 | 3秒 |
| 联锁切换 | 5秒 |
| 故障重试 | 6秒 |

## 通信路径

| 路径 | 允许帧 |
|------|--------|
| CTC主机→联锁主机 | DC2/ACK/RSR/BCC/TSD/ACA |
| 联锁主机→CTC主机 | DC3/ACK/SDI/SDCI/FIR/TSQ/RSR/ACQ |
| CTC主机→联锁备机 | DC2/ACK/RSR |
| 备机→备机 | DC2/DC3/ACK/RSR |

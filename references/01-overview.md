# 系统架构概览

## 快速理解

| 组件 | 说明 |
|------|------|
| **CTC** | 调度集中系统，自律机分主备 |
| **CBI** | 计算机联锁系统，操作表示机分主备 |
| **通信** | 4路交叉通信（CTC主机↔CBI主机/备机，CTC备机↔CBI主机/备机） |

## 日志类型

| 类型 | 文件格式 | 用途 |
|------|----------|------|
| 联锁日志 | `ZLEventsMMDDYY` | 帧解析、设备状态 |
| CTC日志 | `lgxtcidriver_YYYYMMDD_*.log` | 故障诊断 |

## 帧类型速查

| 类型 | 代码 | 方向 | 说明 |
|------|------|------|------|
| DC2 | 0x12 | CTC→联锁 | 连接请求 |
| DC3 | 0x13 | 联锁→CTC | 连接确认 |
| ACK | 0x06 | 双向 | 应答/心跳 |
| SDCI | 0x8A | 联锁→CTC | 站场变化(增量) |
| SDI | 0x85 | 联锁→CTC | 站场完整(全量) |
| FIR | 0x65 | 联锁→CTC | 故障报告 |
| RSR | 0xAA | 双向 | 工作状态 |
| BCC | 0x95 | CTC→联锁 | 按钮命令 |

## 设备类型

| 类型 | 命名 | 示例 |
|------|------|------|
| 道岔区段 | 纯数字 | "64", "110" |
| 信号机 | D+数字 | "D1", "D114" |
| 无岔区段 | 其他 | "DK5", "102G" |

## 时序约束

| 参数 | 值 |
|------|-----|
| ACK超时 | 500ms |
| 最大重试 | 3次 |
| 中断检测 | 1500ms |
| SDIQ间隔 | 24小时 |

## 故障切换规则

- **仅主机-主机故障时切换**
- CTC主机-联锁主机通信中断 > 3秒 → CTC切换
- CTC主机-联锁主机通信中断 > 5秒 → 联锁切换

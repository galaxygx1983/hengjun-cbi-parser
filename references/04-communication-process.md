# 通信流程

## 通信阶段

| 阶段 | 帧序列 | 说明 |
|------|--------|------|
| 连接建立 | DC2 → DC3 | 握手完成，序号初始化 |
| 状态交换 | RSR ↔ RSR | 双方报告主备状态 |
| 初始同步 | SDI → ACK | 联锁发送完整站场 |
| 增量更新 | SDCI → ACK | 持续发送变化 |
| 模式请求 | ACQ → ACA | 自律控制请求 |

## 状态机

| 当前状态 | 触发 | 下一状态 |
|----------|------|----------|
| 中间状态 | ACA 0x55 | 自律控制 |
| 中间状态 | 非常站控按钮 | 非常站控 |
| 中间状态 | ACA超时/0xAA | 中间状态 |
| 自律控制 | 系统故障 | 非常站控 |

## 时序

| 阶段 | 超时 |
|------|------|
| SDI/SDCI → ACK | 500ms |
| ACQ → ACA | 5秒 |
| TSQ → TSD | 500ms |

## 通信内容

| 分类 | 帧 | 方向 |
|------|-----|------|
| 站场状态 | SDI/SDCI/FIR | 联锁→CTC |
| 工作状态 | RSR | 双向 |
| 控制命令 | BCC | CTC→联锁 |
| 时钟同步 | TSQ/TSD | 双向 |
| 心跳 | ACK | 双向 |
| 模式转换 | ACQ/ACA | 双向 |

## BCC命令

| 命令 | 编码 |
|------|------|
| 取消进路 | 4 |
| 故障解锁 | 6 |
| 道岔定位 | 7 |
| 道岔反位 | 8 |
| 引导锁闭 | 16 |

## RSR状态

| 值 | 主/备 | 模式(联锁) | 模式(CTC) |
|----|-------|------------|-----------|
| 0x55 | 主机 | 自律 | 同意 |
| 0xAA | 备机 | 非常站控 | 不同意 |
| 0xCC | 未知 | 中间 | 不确定 |

## 码位表

- **[objects]**: SDCI帧解析，按对象索引
- **[zlobjects]**: SDI帧解析，按字节位置

```
[objects]
#,D1,0        # 信号机D1索引0

[zlobjects]
#,D1,10,0     # D1在字节10，位偏移0
```
